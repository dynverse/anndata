# read_umi_tools

Read a gzipped condensed count matrix from umi_tools.

## Usage

``` r
read_umi_tools(filename, dtype = "float32")
```

## Arguments

- filename:

  File name to read from.

- dtype:

  Numpy data type.

## Examples

``` r
if (FALSE) { # \dontrun{
ad <- read_umi_tools("...")
} # }
```
